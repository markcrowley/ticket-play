Ticket REST API

This is a placeholder page to show you the REST API. Use httpie to post JSON to the application.

To see all tickets, you can do a GET: 

      http GET localhost:9000/v1/tickets

To create new tickets, do a post 

      http POST localhost:9000/v1/tickets creator="Some requestor" description="Some description"

You can always look at the API directly: /v1/tickets
